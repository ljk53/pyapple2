# bin/

ROM and disk files for running the emulator.

## From [Appler](https://github.com/zajo/appler)

Apple ][ emulator for MS-DOS written in 8088 assembly by Emil Dotchevski (zajo).

| File | Description |
|------|-------------|
| `APPLE.ROM` | Apple II ROM image |
| `FLOPPY.ROM` | Disk II controller ROM |
| `APPLER.DSK` | Appler boot disk |
| `MONITOR.LAB` | Monitor label/symbol file |

## Apple II system ROM

| File | Description |
|------|-------------|
| `A2SOFT2.BIN` | Apple II+ ROM with AppleSoft BASIC (12KB, loaded at $D000) |

The `A2SOFT2.BIN` ROM originates from the Apple II+ and contains the Wozniak Monitor and AppleSoft BASIC interpreter. It can also be found in Apple II ROM archives such as [Asimov FTP](https://mirrors.apple2.org.za/ground.icaen.uiowa.edu/MiscInfo/Emulators/apple.roms).
